# Stats

## Information

This module allows your users to show their bot stats.

## Command(s)

`<>` Required `()` Optional

**`/stats`** - Shows the stats of the bot.

## Configuration

The configuration options are located in the `config.js` file under the `stats` section inside the `modules` object.

### Options

```js title="config.js → modules → stats"
stats: {
	enabled: true, // Whether the stats module is enabled
	ephemeral: true, // Whether the bot response should be only visible to the user
},
```
