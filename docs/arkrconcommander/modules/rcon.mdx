# Rcon Commands

## Information

This module allows you to execute commands on one or more servers.

## Command(s)

`<>` Required `()` Optional

**`/rcon command <command> <server|tag>`** - Sends a command to the specified server(s) or tag(s).

## Configuration

There are no configuration options for this module specifically.
